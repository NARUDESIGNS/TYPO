HOW FAST CAN YOU TYPE?

HOW TO PLAY

Well he game is pretty easy to play.
A text is given in a box, alongside a time limit. You are required to type the given text within the given time limit.

There are 3 difficulty options :
Newbie : 1s is allocated to a letter, so for 5 letter word like 'GAMER', 5s is the given time limit to complete typing.

Master : 3s is allocated to 5 letters. So for an 11 letter word, 6s is the given time limit to complete typing. But considering the leftover letter(s), as long as they aren't up to 5 in length, an additional 1s is given, therefore for an 11 letter word, 7s is the given time limit.

Pro : 2s is allocated to 5 letters. So for an 11 letter word, 4s is the given time limit to complete typing. But considering the leftover letter(s), as long as they aren't up to 5 in length, an additional 1s is given, therefore for an 11 letter word, 5s is the given time limit.

For every letter typed, 1pt is being added to the score board located at the top of the page.
